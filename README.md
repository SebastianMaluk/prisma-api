# Backend

## Technologies used

* Typescript
* TRPc
* Prisma
* Fastify
* Zod
* Postgres

# Commands

## start

```cmd
npm run dev
```

## seed

```cmd
npm run db-seed
```


## reset & seed

```cmd
npm run db-reset
```


## push schema to database

```cmd
npx prisma db push
```


## lint

```cmd
npm run lint
```

## prettier-format

```cmd
npm run prettier-format
```
